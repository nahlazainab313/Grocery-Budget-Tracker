About This Project

The Monthly Budget Tracker is a comprehensive financial management application developed using Object-Oriented Programming principles in C++. This project demonstrates the practical application of key OOP concepts including:

  • Inheritance: User base class extended by Income and Expense classes
  • Polymorphism: Virtual functions for flexible object handling
  • Encapsulation: Protected data members with controlled access
  • Templates: Generic functions for data validation
  • Friend Functions: Special access for budget calculations

class Budget : public Income, public Expense {
private:
    double savingsGoal;
    
public:
    double getBalance() const {
        return getTotalIncome() - getTotalExpenses();
    }
    
    double getSavings() const {
        double balance = getBalance();
        return (balance > 0) ? balance : 0;
    }
    
    friend void displayBudgetSummary(const Budget&);
};
